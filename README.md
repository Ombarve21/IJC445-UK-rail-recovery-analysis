# UK Rail Recovery Analysis (2019-2024)

## Project Overview
This project analyses the recovery of Great Britain's rail network following the COVID-19 pandemic. It utilizes data from the Office of Rail and Road (ORR) to visualize:
- Connectivity: Changes in Origin-Destination flows.
- Fares: Analysis of real-term fare changes vs. inflation.
- Recovery Gaps: A sector-by-sector comparison of 2024 volumes vs. 2019 baselines.

## How to Run
1. Clone this repository.
2. Open UK_Rail_Recovery_Analysis.R.
3. Ensure the `data/ folder is in your working directory.
4. Run the script to generate Figures 1-4.

## Dependencies
- R 4.x
- ggplot2, dplyr, readxl, tidyr, scales
